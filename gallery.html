📷 사진 추가 방법

① 사진 파일을 이 HTML 파일과 같은 폴더에 저장하세요.

② gallery.html을 메모장으로 열어서 원하는 칸의 <div class="gallery-placeholder">...</div> 부분을 아래처럼 교체하세요:

<img class="gallery-img" src="파일명.jpg" alt="설명">

③ 새 사진 칸을 추가하려면 아래 코드를 </div><!-- /gallery-grid --> 바로 앞에 붙여넣으세요:

<div class="gallery-item" data-category="stage" data-title="제목" data-caption="설명" onclick="openLightbox(this)"><img class="gallery-img" src="파일명.jpg" alt="설명"><div class="gallery-overlay"><span class="overlay-label">On Stage</span><span class="overlay-title">제목</span></div></div>

※ data-category는 필터에 쓰임: all / rehearsal / stage / behind / event
